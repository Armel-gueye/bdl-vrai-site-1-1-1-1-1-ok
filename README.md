# BinkoO Digital Lab

Site web professionnel de BinkoO Digital Lab, agence spécialisée en IA et Automatisation.

## Technologies

- React
- Vite
- TypeScript
- Tailwind CSS
- Framer Motion

## Installation

```bash
bun install
```

## Développement

```bash
bun run dev
```

## Production

```bash
bun run build
```
